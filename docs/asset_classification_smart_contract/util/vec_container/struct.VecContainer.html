pub struct VecContainer<T> {
    pub values: RefCell<Vec<T>>,
}
Expand description

Contains a RefCell that manages a Vec of Clone-implementing values. Allows for the internal values to be mutated without declaring the value as mutable.

Fields§

§values: RefCell<Vec<T>>

The inner values of the container that are manipulated by the various implementation functions.
