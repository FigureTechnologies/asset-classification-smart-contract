pub fn for_asset_event<T1: Into<String>, T2: Into<String>>( event_type: EventType, asset_type: T1, scope_address: T2, ) -> Self

Certain events like onboard_asset require a standard set of event types. This is a constructor for the struct that includes those values to facilitate the process of generating all events.

§Parameters
  • event_type All events should denote their type for external consumers of Provenance Blockchain Event Stream, so this value is required for any new instance and appends the name of the event with the key of ASSET_EVENT_TYPE_KEY.
  • asset_type A unique key for an AssetDefinitionV3 that uses the key ASSET_TYPE_KEY.
  • scope_address A unique key for an AssetDefinitionV3 that uses the key ASSET_SCOPE_ADDRESS_KEY.
