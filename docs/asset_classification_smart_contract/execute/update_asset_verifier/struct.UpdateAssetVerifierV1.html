Expand description

A transformation of ExecuteMsg::UpdateAssetVerifier for ease of use in the underlying update_asset_verifier function.

Parameters

  • asset_type The unique identifier for the target AssetDefinitionV3, keyed on its asset_type property that the target verifier detail lives in, in its verifiers property.
  • verifier The verifier detail that will be updated. All values within this provided struct will replace the existing detail on the target AssetDefinitionV3.
